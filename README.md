# 🏗 SkyDealer — Backend (API & Database Layer)

Цей репозиторій містить серверну логіку та опис архітектури даних для платформи SkyDealer.

## 🏛 Архітектура
На поточному етапі розробки використовується **Mock-backend** архітектура. Уся бізнес-логіка тимчасово обробляється на стороні клієнта (Frontend) за допомогою Context API та LocalStorage.

## 🗺 План розвитку (Roadmap)
1. Створення REST API на **Node.js (Express)**.
2. Підключення нереляційної бази даних **MongoDB** для зберігання літаків, користувачів та заявок.
3. Налаштування справжньої JWT-авторизації.
