# 🛫 SkyDealer — Backend (REST API & Database Layer)

Цей репозиторій містить серверну частину (блок API та логіку роботи з базою даних) для курсового проєкту **SkyDealer** — платформи з продажу та лізингу приватної авіації.

## 🛠 Стек технологій
* **Платформа:** Node.js
* **Фреймворк:** Express.js
* **База даних:** MongoDB (хмарний кластер Atlas) + Mongoose
* **Аутентифікація:** JSON Web Tokens (JWT)
* **Безпека:** bcryptjs (хешування паролів), CORS

## ⚙️ Реалізований функціонал
На даному етапі повністю реалізовано повноціннуверну архітектуру:
1. **REST API:** Налаштовані маршрути для виконання CRUD-операцій з літаками (створення, читання, оновлення, видалення).
2. **База даних:** Підключена нереляційна БД MongoDB для надійного зберігання списку літаків, користувачів та заявок.
3. **Авторизація:** Реалізована система реєстрації та логіну з видачею JWT-токенів та надійним хешуванням паролів користувачів.
4. **Seeding:** Створено скрипт для автоматичного початкового наповнення бази даних каталогом літаків.

## 🚀 Як запустити проєкт локально

1. **Склонуйте репозиторій:**
   ```bash
   git clone <посилання-на-твій-репозиторій>
2. Встановіть залежності:
    npm install
3. Налаштуйте змінні оточення:
    Створіть файл .env у корені проєкту та додайте наступні ключі:

    Фрагмент коду
    PORT=5000
    MONGO_URI=ваше_посилання_на_mongodb_atlas
    JWT_SECRET=ваш_секретний_ключ_для_токенів
4. Запустіть сервер:
    Для режиму розробки (з автоперезавантаженням): npm run dev
    Для продакшену: npm start

    Сервер буде запущено за адресою: http://localhost:5000
